Come usare gitHub

Iniziamo andare sul sito, e dopo essere entrati premiamo sul tasto verde New diamo un nome 'UsareGitHub' -lasciamo spuntato 'public' -spuntiamo Add a README file -Premiamo Create repository

Adesso voglio copiare il contenuto vado sul mio progetto in alto a destra tasto <> Code e scelgo Clone HTTPS e copio il link. Andiamo sulla directory 'gitHub_esempio' e scriviamo: git clone https://github.com/skywall68/UsareGitHub.git Questo comando mi crea all'interno della cartella gitHub_esempio una nuova cartella chiamata 'UsareGitHub' Al suo interno abbiamo il file README.md e una cartella nascosta chiamata .git che non vediamo. Adesso prendiamo i nostri file e li portiamo dentro la cartella 'UsareGitHub' Andiamo su 'UsareGitHub' con cd e poi per vedere i file scriviamo ls Vediamo i file spostati.

Il secondo comando che usiamo è vedere se ci sono i dati usando gitHub: git status Abbiamo 'Untracked' tracciamo i nostri files

Il terzo comando è aggiungere i files uno per uno: git add index.html git add script.js Se abbiamo una directory con molti files all'interno: git add directory/. Il punto (.) è * copia tutti i files ... tutti i files (non ho incluso leggimi.txt in questo comando) adesso posso vedere lo stato: git status Vedo che leggimi.txt non è stato tracciato e per essere poi 'committed' impiegati nel loro utilizzo.

Il quarto passaggio è quello di renderli utilizzabili con gitHub: git commit -m "mio messaggio come:'ho iniziato questo nuovo progetto'"

Il quinto passaggio bisogna far sapere a GitHub il cambiamento con: git push Se ho già un progetto con file e cartelle quando arrivo a questo punto devo trovarmi sulla directory(master) e scrivere: $ git push --set-upstream https://github.com/skywall68/indovinaNumero.git master Questo mi serve per inviare i dati a GitHub

Adesso voglio recuperare il progetto in un altra cartella chiamata recupero, vado sulla mia pagina di gitHub, premo il tasto verde copio l'indirizzo http sotto clone: https://github.com/skywall68/UsareGitHub.git dentro la cartella 'recupero' scrivo: git clone https://github.com/skywall68/UsareGitHub.git

Mettiamo che adesso ho modificato il file E VOGLIO AGGIORNARLO: Sempre nella cartella 'main' scrivo: git status Visualizzo i files modificati. Adesso aggiorno GitHub con il scrivere la modifica del singolo file: git add index.html oppure se ho modificato diversi files scrivo: git add . Dopo questo scrivo: git commit -m "mio messaggio" git push A questo punto il codice è stato modificato su GitHub.
