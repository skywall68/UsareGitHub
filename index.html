Come usare gitHub

Iniziamo andare sul sito, e dopo essere entrati premiamo sul tasto verde New diamo un nome 'UsareGitHub' -lasciamo spuntato 'public' -spuntiamo Add a README file -Premiamo Create repository

Adesso voglio copiare il contenuto vado sul mio progetto in alto a destra tasto <> Code e scelgo Clone HTTPS e copio il link. Andiamo sulla directory 'gitHub_esempio' e scriviamo: git clone https://github.com/skywall68/UsareGitHub.git Questo comando mi crea all'interno della cartella gitHub_esempio una nuova cartella chiamata 'UsareGitHub' Al suo interno abbiamo il file README.md e una cartella nascosta chiamata .git che non vediamo. Adesso prendiamo i nostri file e li portiamo dentro la cartella 'UsareGitHub' Andiamo su 'UsareGitHub' con cd e poi per vedere i file scriviamo ls Vediamo i file spostati.

Il secondo comando che usiamo è vedere se ci sono i dati usando gitHub: git status Abbiamo 'Untracked' tracciamo i nostri files

Il terzo comando è aggiungere i files uno per uno: git add index.html git add leggimi.txt ... tutti i files
